# Changelog

## [1.0.0] - 2026-07-09

### Added
- Основной чат с AI (Groq, DeepSeek, OpenRouter, HuggingFace)
- Векторный поиск с pgvector
- Гибридный поиск (векторный + ключевой)
- Веб-поиск (DuckDuckGo, SearXNG)
- Загрузка документов (PDF, DOCX, XLSX, TXT)
- Интеграция с Google Drive
- Автоматическая синхронизация по расписанию
- Админ-панель
- Копирование и скачивание ответов
- Поддержка сессий чата
- TypeScript типизация
- Тесты (Vitest)
- Docker support
- CI/CD (GitHub Actions)

### Features
- Fallback chain для AI провайдеров
- Автоматическое определение типа документа
- Автоматическое определение выдающего органа
- Chunking с overlap
- Markdown rendering
- Mobile responsive design
